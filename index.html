This GitHub repository currently contains only documentation. The application code has not yet been committed.

Budbase is planned as a modern agricultural data management platform for precision farming, vineyard management, and crop monitoring.

Planned Technology Stack
  • Frontend: React 18 + TypeScript + Vite + Tailwind CSS
  • Authentication: Supabase
  • Backend: Google Cloud Run (Express + TypeScript)
  • Storage: Google Cloud Storage
  • Maps: Mapbox GL JS + Turf.js
Next Steps

To get this project running:

  • Push the frontend/ directory with React code to the repository
  • Include package.json and other configuration files
  • Add any backend code if available
  • The Replit environment will then be configured automatically

Check the replit.md file for more details about the expected setup.
